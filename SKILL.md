# AI YouTube Thumbnail Generator

An intelligent YouTube thumbnail maker that uses AI to create compelling thumbnails from video content, text prompts, or uploaded images.

## Features

- **AI-Powered Generation**: Uses Google's Gemini AI to create thumbnails from text descriptions
- **Video Frame Extraction**: Generate thumbnails from video files by extracting key frames
- **Image Editor**: Built-in editor for customizing generated thumbnails
- **Template Library**: Pre-designed templates for various YouTube niches
- **Background Removal**: Remove and replace backgrounds automatically
- **User Authentication**: Secure user management with Supabase
- **Usage Analytics**: Track thumbnail generation usage and performance
- **Subscription Plans**: Multiple pricing tiers for different user needs

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **AI/ML**: Google Gemini AI API
- **Backend**: Supabase (Auth, Database, Storage)
- **Styling**: Tailwind CSS, Framer Motion
- **Deployment**: Railway, Cloudflare Pages
